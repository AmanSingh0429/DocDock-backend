# HTTP Response Status Codes

## Success
- 200 OK
- 201 Created
- 204 No Content

## Redirection
- 301 Moved Permanently
- 302 Found
- 304 Not Modified

## Client Error
- 400 Bad Request
- 401 Unauthorized
- 403 Forbidden
- 404 Not Found
- 409 Conflict
- 422 Unprocessable Entity
- 429 Too Many Requests

## Server Error
- 500 Internal Server Error
- 502 Bad Gateway
- 503 Service Unavailable
- 504 Gateway Timeout
